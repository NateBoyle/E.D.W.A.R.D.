# E.D.W.A.R.D.
E.D.W.A.R.D. (the Electronic Data Writer Analyzer Reporter and Documenter) takes in a slightly formatted but still simple trial balance CSV file and produces the user's choice of balance sheet and income statement and/or various visualizations of the financial data in the trial balance.
